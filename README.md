# Getting Started
You can use starback.js directly with `<script>` tag or import either.
- With <script> tag
  ```html
  <script src="PATH_TO_DIST_FOLDER/starback.js"></script>
  ```
- With import
  ```js
  import Starback from 'starback'
  ```
